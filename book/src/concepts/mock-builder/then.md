# Then

`Then` is a builder used to build responses.

### Body methods:
- `body()` *(primary)*
- `empty()`
- `bytes()`
- `bytes_stream()`
- `text()`
- `text_stream()`
- `json()`
- `json_lines_stream()`
- `pb()`
- `pb_stream()`

### Headers method:
- `headers()`

### Status methods:
- `status()` *(primary)*
- `message()`
- `error()`
- `ok()`
- `bad_request()`
- `internal_server_error()`
- `not_found()`
